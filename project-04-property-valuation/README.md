# 📊 Property Valuation & ROI Engine
**Sector:** Real Estate | **Build Time:** ~3 hours | **Difficulty:** ⭐⭐⭐

## One-Line Pitch
> Automatically generates investor-grade property ROI reports by pulling live comparable sales data and running AI analysis.

## ROI
- 3–4 Hours Analyst Work → 30 Seconds
- \+ Saved Per Report Generated

## Stack
\
8n\ · \Google Maps API\ · \OpenAI GPT-4\ · \Google Sheets\ · \Slack\

## Setup
1. Copy \.env.example\ to \.env\ and fill in your API keys.
2. Import \workflow.json\ into your n8n instance.
3. Configure your Google Sheets template for report output.
4. Activate the workflow.
