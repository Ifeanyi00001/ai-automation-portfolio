# 🔍 Document Risk Auditor (Due Diligence)
**Sector:** Enterprise / M&A | **Build Time:** ~4 hours | **Difficulty:** ⭐⭐⭐⭐⭐

## One-Line Pitch
> Processes entire M&A data rooms, flagging CRITICAL/HIGH/MEDIUM risks across every document with exact clause references — in minutes not weeks.

## ROI
- 100% Data Room Coverage (Zero Missed Risks)
- Weeks of Manual Review → Hours of AI Processing

## Stack
\
8n\ · \LlamaParse\ · \Pinecone\ · \Claude 3.5 Sonnet\ · \OpenAI\ · \Slack\

## Setup
1. Copy \.env.example\ to \.env\ and fill in your API keys.
2. Import \workflow.json\ into your n8n instance.
3. Configure your document source folder and Pinecone index.
4. Activate the workflow.
