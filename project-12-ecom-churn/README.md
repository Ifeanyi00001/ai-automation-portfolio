# 🛍️ E-commerce Churn Predictor
**Sector:** Shopify / E-commerce | **Build Time:** ~3 hours | **Difficulty:** ⭐⭐⭐⭐

## One-Line Pitch
> Uses RFM (Recency, Frequency, Monetary) analysis to identify at-risk customers in real-time and triggers automated win-back campaigns.

## ROI
- \,500/month Revenue Leakage Recovered
- 5x Cheaper to Retain Than Acquire a New Customer

## Stack
\
8n\ · \Shopify\ · \Claude 3.5 Sonnet\ · \Klaviyo\ · \Slack\

## Setup
1. Copy \.env.example\ to \.env\ and fill in your API keys.
2. Import \workflow.json\ into your n8n instance.
3. Connect your Shopify store and Klaviyo account.
4. Activate the workflow — it runs on a daily schedule.
