## Setup
1. Copy \.env.example\ to \.env\ and fill in your API keys.
2. Import \workflow.json\ into your n8n instance.
3. Configure your stock watchlist in the workflow settings.
4. Activate the workflow — briefing auto-delivers at 6AM daily.
