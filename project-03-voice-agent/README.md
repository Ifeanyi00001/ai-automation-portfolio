# 🏠 AI Property Concierge (Voice Agent)
**Sector:** Real Estate | **Build Time:** ~4 hours | **Difficulty:** ⭐⭐⭐⭐⭐

## One-Line Pitch
> Calls new property leads within 10 seconds of form submission, qualifies them via AI, and auto-books viewings 24/7.

## ROI
- Replaces 1 Full-time Receptionist
- Speed-to-Lead: Under 10 Seconds (24/7 Operation)

## Stack
\
8n\ · \Vapi\ · \Claude 3.5 Sonnet\ · \HubSpot\ · \Calendly\ · \Slack\ · \Gmail\

## Setup
1. Copy \.env.example\ to \.env\ and fill in your API keys.
2. Import \workflow.json\ into your n8n instance.
3. Configure your Vapi assistant with the system prompt in the documentation.
4. Activate the workflow.
