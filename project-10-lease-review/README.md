# 🏢 Lease Review & Tenant Onboarding Pipeline
**Sector:** Legal Ops / Property | **Build Time:** ~3.5 hours | **Difficulty:** ⭐⭐⭐⭐

## One-Line Pitch
> Extracts key terms from lease PDFs using LlamaParse, flags risk clauses with Claude, and routes to DocuSign for e-signature — all automatically.

## ROI
- 3 Hours Property Manager Admin → 90 Seconds
- Faster Occupancy Rates and Zero Signature Errors

## Stack
\
8n\ · \LlamaParse\ · \Claude 3.5 Sonnet\ · \DocuSign\ · \HubSpot\ · \Slack\

## Setup
1. Copy \.env.example\ to \.env\ and fill in your API keys.
2. Import \workflow.json\ into your n8n instance.
3. Configure your DocuSign template with signature anchor tags.
4. Activate the workflow.
